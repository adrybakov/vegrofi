*******************************************
Verificator for GROGU spin Hamiltonian file
*******************************************

To verify the spin Hamiltonian file:

*   (optional) Create virtual environment

    .. code-block:: bash
        python3.13 -m venv .venv

*   (optional) Activate virtual environment

    .. code-block:: bash
        source .venv/bin/activate

* Install the tool

    .. code-block:: bash
        pip install vegrofi

* Run the Verificator

    .. code-block:: bash
        vegrofi <filename>

* Enjoy
